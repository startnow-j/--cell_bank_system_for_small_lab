## 一、系统概述

### 1.1 系统简介

细胞库管理系统是一个专业的生物样本库管理平台，主要用于管理冻存细胞的入库、存储、出库和统计等全流程操作。系统支持多用户协作，通过角色权限控制确保数据安全。

### 1.2 主要功能

| 功能模块 | 功能描述 |
|---------|---------|
| 仪表盘 | 数据概览、快捷操作入口 |
| 存储位置 | 管理冰箱、架子、盒子的层级结构 |
| 细胞入库 | 单个入库、批量入库、入库记录查询 |
| 库存查询 | 按条件检索库存、查看存储位置 |
| 细胞出库 | 单个出库、整盒出库、出库记录查询 |
| 统计报表 | 数据统计、可视化图表 |
| 用户管理 | 用户增删改查、角色分配（仅管理员） |

### 1.3 系统特点

- **响应式设计**：支持PC端和移动端访问
- **权限分级**：管理员、操作员、观察员三级权限
- **批量操作**：支持Excel批量导入入库数据
- **可视化统计**：饼图、折线图、柱状图展示数据

## 二、登录系统

### 2.1 访问系统

在浏览器中输入系统地址，进入登录页面。

### 2.2 输入账户信息

在登录页面输入：
- **邮箱**：您的账户邮箱
- **密码**：账户密码

点击「登录」按钮进入系统。

### 2.3 修改密码

登录后，在左侧边栏点击用户信息旁的🔑图标，可修改密码：
1. 输入旧密码
2. 输入新密码（至少6位）
3. 确认新密码
4. 点击「确认修改」

> **提示**：密码修改成功后需使用新密码重新登录。

## 四、功能模块详解

### 4.1 仪表盘

仪表盘提供系统数据概览和快捷操作入口。

#### 统计卡片

| 卡片 | 说明 |
|------|------|
| 冰箱数量 | 系统中配置的冰箱总数 |
| 在库细胞 | 当前库存的细胞管数 |
| 已取出 | 已出库的细胞管数 |
| 用户数量 | 系统用户总数 |

#### 快捷操作

提供四个常用功能的快捷入口：
- **细胞入库**：跳转到入库页面
- **查看库存**：跳转到库存查询页面
- **细胞出库**：跳转到出库页面
- **统计报表**：跳转到统计报表页面

---

### 4.2 存储位置

管理细胞的物理存储位置，采用三级层级结构：**冰箱 → 架子 → 盒子**。

#### 层级结构说明

```
冰箱(Freezer)
├── 架子(Rack) 1
│   ├── 盒子(Box) A
│   ├── 盒子(Box) B
│   └── ...
├── 架子(Rack) 2
│   └── ...
└── ...
```

#### 操作说明

**添加冰箱**：
1. 点击「添加冰箱」按钮
2. 输入冰箱名称
3. 点击保存

**添加架子**：
1. 点击冰箱名称展开
2. 点击「添加架子」
3. 输入架子名称和容量
4. 点击保存

**添加盒子**：
1. 点击架子展开
2. 点击「添加盒子」
3. 输入盒子名称、行数、列数
4. 点击保存

**删除操作**：
- 只能删除没有存储细胞的盒子
- 只能删除没有盒子的架子
- 只能删除没有架子的冰箱

> **注意**：删除操作不可撤销，请谨慎操作。

### 4.3 细胞入库

细胞入库模块提供三种功能：新增入库、批次入库、入库记录。

#### 4.3.1 新增入库（单个入库）

适用于少量细胞的入库操作。

**操作步骤**：

1. **填写细胞信息**
   
   | 字段 | 是否必填 | 说明 |
   |------|---------|------|
   | 细胞编号 | 否 | 自定义编号，如CELL-001 |
   | 细胞名称 | 是 | 如HEK293 |
   | 细胞类型 | 是 | 如贴壁细胞、悬浮细胞 |
   | 代次 | 是 | 如P5，未知填"未知" |
   | 冻存数量 | 是 | 冻存的管数 |
   | 冻存日期 | 是 | 格式：YYYY-MM-DD |
   | 冻存液 | 否 | 冻存保护液配方 |
   | 供体信息 | 否 | 来源信息 |
   | 培养条件 | 否 | 培养基、温度等 |
   | 操作人 | 否 | 操作者姓名 |
   | 备注 | 否 | 其他备注 |

2. **选择存储位置**
   
   - 依次选择：冰箱 → 架子 → 盒子
   - 系统显示盒子的位置网格
   - 点击空闲位置进行选择
   - 已占用位置显示为灰色，不可选择
   - 选择的数量必须与冻存数量一致

3. **确认入库**
   
   - 检查信息无误后点击「确认入库」
   - 系统提示成功后自动刷新

#### 4.3.2 批次入库（仅管理员）

适用于大批量细胞数据的导入。

**操作步骤**：

1. **下载模板**
   
   点击「下载模板」按钮，下载CSV模板文件。

2. **填写模板**
   
   | 列名 | 是否必填 | 说明 |
   |------|---------|------|
   | 细胞编号 | 否 | 自定义编号 |
   | 细胞名称* | 是 | 细胞名称 |
   | 细胞类型* | 是 | 细胞类型 |
   | 代次* | 是 | 代次 |
   | 冻存数量(管)* | 是 | 冻存管数 |
   | 冻存日期* | 是 | YYYY-MM-DD格式 |
   | 冻存液 | 否 | 冻存液配方 |
   | 供体信息 | 否 | 来源信息 |
   | 操作人 | 否 | 操作者姓名 |
   | 备注 | 否 | 其他备注 |
   | 冰箱名称* | 是 | 需与系统中名称完全一致 |
   | 架子名称* | 是 | 需与系统中名称完全一致 |
   | 盒子名称* | 是 | 需与系统中名称完全一致 |
   | 位置* | 是 | 多个用逗号分隔，如A1,B2,C3 |

3. **上传文件**
   
   - 点击上传区域或将文件拖入
   - 支持CSV、XLS、XLSX格式

4. **校验数据**
   
   系统自动校验文件，显示：
   - 有效数据数量
   - 错误数据数量
   - 错误详情（行号+错误原因）

5. **修正错误**
   
   - 修正Excel中的错误数据
   - 重新上传

6. **开始入库**
   
   校验通过后点击「开始入库」，系统批量创建数据。

**常见校验错误**：

| 错误类型 | 说明 | 解决方法 |
|---------|------|---------|
| 必填字段为空 | 未填写必填项 | 补充必填字段 |
| 位置数量不一致 | 位置数量≠冻存数量 | 确保两者一致 |
| 冰箱/架子/盒子不存在 | 名称与系统不匹配 | 检查名称是否完全一致 |
| 位置已被占用 | 指定位置已有细胞 | 更换其他位置 |
| 日期格式错误 | 日期格式不正确 | 使用YYYY-MM-DD格式 |

#### 4.3.3 入库记录

查看历史入库记录。

**查询功能**：
- 支持按细胞名称、批次编号搜索
- 支持分页浏览

**操作**：
- 点击记录查看详情
- 点击「导出」可导出Excel文件

### 4.4 库存查询

查询当前在库的细胞信息。

#### 查询条件

| 条件 | 说明 |
|------|------|
| 细胞名称 | 模糊匹配 |
| 细胞类型 | 下拉选择 |
| 冰箱 | 下拉选择 |
| 批次编号 | 精确匹配 |

#### 查询结果

显示符合条件的细胞列表，包含：
- 细胞编号
- 细胞名称
- 细胞类型
- 代次
- 冻存日期
- 存储位置（冰箱/架子/盒子/位置）
- 操作（查看详情）

#### 位置可视化

点击某条记录可查看位置详情，系统用网格图展示盒子内存储情况：
- 绿色：当前选中的细胞
- 灰色：其他已占用位置
- 白色：空闲位置

### 4.5 细胞出库

细胞出库模块提供两种功能：取出细胞、出库记录。

#### 4.5.1 取出细胞

**单个取出**：

1. **查找细胞**
   
   - 在搜索框输入细胞名称或编号
   - 系统显示匹配的批次列表

2. **选择批次**
   
   点击要取出的批次，系统显示：
   - 批次基本信息（名称、类型、代次、冻存日期等）
   - 该批次所有细胞的位置分布

3. **选择位置**
   
   - 点击网格中选择要取出的位置
   - 已取出的位置显示为灰色

4. **填写出库信息**
   
   | 字段 | 是否必填 | 说明 |
   |------|---------|------|
   | 出库原因 | 是 | 如实验使用、转移等 |
   | 操作人 | 否 | 操作者姓名 |
   | 备注 | 否 | 其他备注 |

5. **确认出库**
   
   点击「确认出库」完成操作。

#### 4.5.2 整盒出库（仅管理员）

适用于将整盒细胞一次性出库。

**操作步骤**：

1. 依次选择：冰箱 → 架子 → 盒子
2. 系统显示盒子内所有细胞
3. 确认后整盒出库

> **注意**：整盒出库后，盒内所有细胞状态变为"已取出"，盒子保留但为空。

#### 4.5.3 出库记录

查看历史出库记录。

**查询功能**：
- 支持按细胞名称搜索
- 支持按操作人筛选
- 支持分页浏览

**显示信息**：
- 细胞信息
- 出库时间
- 出库原因
- 操作人

提供数据统计和可视化图表。

#### 4.6.1 统计卡片

| 卡片 | 说明 |
|------|------|
| 总库存 | 当前在库细胞总数 |
| 批次总数 | 入库批次总数 |
| 本月入库 | 本月入库的细胞管数 |

#### 4.6.2 图表（仅PC端）

- 显示各类型细胞的占比

**快捷选项**：
- 本月
- 上月
- 近3个月
- 今年
- 去年

**自定义范围**：
- 点击日历选择开始日期
- 点击日历选择结束日期
- 点击确定应用筛选

## 六、常见操作流程

### 6.1 新细胞入库流程

```
1. 准备工作
   ├── 确认存储位置（冰箱/架子/盒子）
   └── 准备细胞信息（名称、类型、代次等）

2. 选择入库方式
   ├── 单个入库（≤10管）
   │   ├── 进入「细胞入库」→「新增入库」
   │   ├── 填写细胞信息
   │   ├── 选择存储位置
   │   └── 确认入库
   │
   └── 批量入库（>10管）
       ├── 进入「细胞入库」→「批次入库」
       ├── 下载模板
       ├── 填写Excel数据
       ├── 上传并校验
       └── 确认入库

3. 完成入库
   └── 在「入库记录」中确认数据已入库
```

### 6.2 细胞取出流程

```
1. 查找细胞
   ├── 进入「细胞出库」→「取出细胞」
   └── 搜索细胞名称或编号

2. 选择批次
   └── 点击要取出的批次

3. 选择位置
   └── 在位置网格中点击要取出的管

4. 填写信息
   ├── 输入出库原因
   ├── 输入操作人（可选）
   └── 输入备注（可选）

5. 确认出库
   └── 点击「确认出库」
```

### 6.3 查询库存流程

```
1. 进入库存查询
   └── 点击侧边栏「库存查询」

2. 设置查询条件
   ├── 输入细胞名称（模糊搜索）
   ├── 选择细胞类型
   ├── 选择冰箱
   └── 输入批次编号

3. 查看结果
   ├── 列表显示匹配的细胞
   └── 点击查看位置详情

4. 导出数据（可选）
   └── 点击「导出」下载Excel
```

### 6.4 新增存储位置流程

```
1. 进入存储位置管理
   └── 点击侧边栏「存储位置」

2. 添加冰箱
   ├── 点击「添加冰箱」
   ├── 输入冰箱名称
   └── 保存

3. 添加架子
   ├── 点击冰箱展开
   ├── 点击「添加架子」
   ├── 输入架子名称和容量
   └── 保存

4. 添加盒子
   ├── 点击架子展开
   ├── 点击「添加盒子」
   ├── 输入盒子名称、行数、列数
   └── 保存

5. 开始使用
   └── 入库时可选择新添加的位置
```

## 七、注意事项

### 7.1 数据安全

1. **定期备份**：建议定期导出数据进行备份
2. **账户安全**：不要共享账户，定期修改密码
3. **权限管理**：按需分配最小权限

### 7.2 操作规范

1. **入库前检查**：确认存储位置名称与Excel中一致
2. **批量入库**：先少量测试，确认无误后再大批量导入
3. **出库记录**：务必填写出库原因，便于追溯

### 7.3 系统限制

1. **删除限制**：有数据的盒子/架子/冰箱不能删除
2. **位置唯一性**：一个位置只能存储一管细胞
3. **名称唯一性**：用户邮箱不能重复

### 7.4 移动端使用

1. **功能限制**：统计报表模块不显示图表
2. **建议使用电脑端**进行大批量操作
3. **横屏使用**可获得更好的显示效果

## 八、常见问题

### Q1: 忘记密码怎么办？

请联系系统管理员重置密码。

### Q2: 批量入库时提示"位置已被占用"？

检查Excel中填写的位置是否已被其他细胞占用。可以在库存查询中确认该位置状态。

### Q3: 批量入库时提示"冰箱不存在"？

确保Excel中的冰箱名称与系统中的名称**完全一致**，包括空格和大小写。

### Q4: 如何查看某个批次的详细信息？

在「入库记录」中搜索批次，点击记录可查看详情，包括所有细胞的位置分布。

### Q5: 误操作出库了怎么办？

出库操作不可撤销。如果误操作，需要重新入库。

### Q6: 为什么有些菜单看不到？

不同角色看到的功能菜单不同，请参考「角色与权限」章节。

### Q7: 移动端为什么看不到图表？

为了确保移动端的性能和用户体验，统计报表模块在移动端只显示数据卡片，不显示图表。请在电脑端查看完整图表。

### Q8: 如何导出数据？

- 入库记录：在入库记录页面点击「导出」
- 库存查询：在库存查询页面点击「导出」
- 出库记录：在出库记录页面点击「导出」

### Q9: 批量入库模板可以修改列名吗？

不建议修改列名。系统会自动识别标准列名，修改后可能导致识别失败。

### Q10: 如何添加新的细胞类型？

细胞类型为自由输入，入库时直接输入新类型即可。建议保持类型名称的一致性。
